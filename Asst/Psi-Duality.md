# Theorem 11: State Transition & Limit Symmetry
**Axiom:** The boundary between the frozen C and active Q sectors is a limit transition.
- **Equation:**
$$\lim_{\epsilon \to 0^-} \Psi_{Phys} = \Psi(-0) \longleftrightarrow \lim_{\epsilon \to 0^+} \Psi_{Inf} = \Psi(0+)$$
- **Logic:** Information is neither created nor destroyed during the jump across the Omnium Core ($S=0$).
